# ChromaSound

一个基于摄像头手势识别的实时音乐绘画交互系统。

用户通过摄像头捕获手部动作，利用手势识别控制画布绘制，同时将手部运动映射到音乐生成过程，实现视觉绘画、音乐演奏和粒子特效的实时互动。

---

## 项目介绍

ChromaSound 是一个结合计算机视觉、音乐生成和实时图形效果的交互式艺术系统。

程序运行后会显示：

* 主画布区域
* 右下角实时摄像头窗口

用户可以通过单手手势进行绘画和音乐控制：

* 食指控制画笔绘制轨迹
* 手部移动控制音符、和弦以及音量
* 不同颜色对应不同的音乐组合
* 声音触发时同步生成视觉反馈效果

---

# 功能特点

## 1. 手势绘画

* 使用摄像头实时捕获手部动作
* 使用 MediaPipe Hand Landmarker 进行单手识别
* 当检测到食指伸出时：

  * 指尖作为画笔
  * 在画布中绘制轨迹

---

## 2. 手势音乐控制

### 水平方向移动

控制：

* 主旋律乐器
* 和弦乐器

---

### 垂直方向移动

控制：

* 和弦触发

---

### 空间位置映射

画布中的不同位置对应不同音高。

用户可以通过移动手指位置演奏不同音符。

---

### 速度控制音量

根据手部移动速度动态调整音量：

* 移动速度越快，声音越强
* 移动速度越慢，声音越弱

---

## 3. 颜色与音乐映射

用户可以通过：

> 食指与拇指触碰

切换画笔颜色。

颜色循环：

```
红色 → 蓝色 → 绿色 → 红色
```

每种颜色对应不同的主乐器组合。

和弦也会同步对应当前颜色。

---

## 4. 实时视觉反馈

每次触发声音时：

### 背景光晕

* 在画布随机位置生成
* 颜色与当前画笔颜色一致
* 光晕会逐渐缩小并消失

### 指尖粒子效果

* 指尖附近生成粒子群
* 粒子颜色与画笔颜色一致
* 粒子随后逐渐消失

---

## 5. 录音与结果保存

程序运行过程中：

按下：

```
S
```

开始录制。

再次按下：

```
S
```

结束录制。

结束后保存：

* 本次交互过程的音频文件
* 结束时的画面截图

# 项目结构

```
ChromaSound
│
├── proj.py                 # 主程序入口
├── audio.py                # 音频播放、音符和和弦处理、录音保存
├── music_engine.py         # 音乐逻辑控制
├── gesture.py              # 手势判断相关功能
├── motion.py               # 手部运动速度计算与平滑处理
├── trajectory.py           # 手势运动轨迹分析
├── particles.py            # 粒子效果与视觉反馈
├── rhythm.py               # 节奏控制
├── scale.py                # 音阶与音高计算
├── state.py                # 程序状态管理
│
├── hand_landmarker.task    # MediaPipe 手部识别模型文件
│
├── requirements.txt        # Python依赖列表
│
└── README.md               # 项目说明文档
```

# 安装依赖

克隆项目：

```bash
git clone https://github.com/ling775/ChromaSound.git
```

进入项目目录：

```bash
cd ChromaSound
```

安装 Python 依赖：

```bash
pip install -r requirements.txt
```

---

# 启动方式

运行：

```bash
python proj.py
```

程序启动后：

1. 摄像头窗口开始捕获手势
2. 使用食指在画布中绘制
3. 移动手势控制音乐
4. 使用手指触碰切换颜色
